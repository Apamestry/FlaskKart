# FlaskKart

A modern E-Commerce website built using:

- Flask
- MySQL (Docker)
- SQLAlchemy
- Bootstrap 5

## Features

- User Authentication
- Product Catalog
- Product Search
- Shopping Cart
- Checkout
- Order History
- Admin Dashboard

## Tech Stack

- Python
- Flask
- MySQL
- Docker
- SQLAlchemy
